汽车项目回顾

发表于 2019-03-06 | 更新于 2019-04-08 | 分类于 项目回顾

项目概述

项目内容包括车联网管理平台、分时租赁平台、互联网车辆营销平台三个板块的业务。车联网通过对接TSP或直接对接车辆TBOX实现车辆的联网，对车联设备提供各类增值服务；分时租赁平台是在车联网平台的基础上，实现专车、分时租赁、机构用车等多种分时用车的服务；互联网车辆营销是基于传统4S点的汽车销售模式，基于互联网营销模式构建的新型车辆销售模式，提供潜客拓展、售前服务、预约用车、销售服务、售后服务等功能。

项目基于Spring Boot/Cloud构建的微服务框架实现业务的原子化封装和集群化部署。

主要技术

  • Java + Spring MVC + Spring Boot + Spring Cloud
  • Spring JPA + Hibernate
  • Flume
  • Netty
  • Hazelcast 、Redis
  • MySQL + HBase + Impala
  • ArgularJS + Bootstrape
  • APP（Android + iOS）
  • 微信公众号
