电影项目回顾

发表于 2019-03-05 | 更新于 2019-04-08 | 分类于 项目回顾

项目概述

对全国所有影票实时收取每个张票的上报信息，针对影院、网售商、每日合报数据进行数据实时校验、交叉校验，提供实时影票的二维码扫描验真服务，提供数据统计分析服务，以及数据管理业务需要的各种功能。项目涉及Openstack云平台构建、Hadoop大数据平台构建、微服务数据处理平台、数据接入网关集群平台、数据服务平台、APP数据查看、微信数据报告和监控等。

主要技术

  • Openstack
  • Hadoop
  • Java + Spring MVC + Spring Boot + Spring Cloud
  • Spring JPA + Hibernate
  • Flume
  • Netty
  • Hazelcast
  • MySQL + HBase + Impala
  • ArgularJS + Bootstrape
  • C#
  • APP（Android + iOS）
  • 微信公众号
  • 数据服务网关
